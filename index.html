Frontend pronto para Vercel

Gerar e validar keys com backend KeyAuth em servidor proprio.

Este frontend foi feito para deploy direto no Vercel. A geracao da key passa pela API serverless e usa variaveis de ambiente, sem expor sellerkey no navegador.

Abrir Painel Deploy no Vercel

Arquitetura recomendada

1. Frontend no Vercel

2. KeyAuth source (PHP + MySQL + Redis) em VPS/servidor

3. Painel usa:

/api/generate-key para criacao

/api/validate-key para validacao online

Seguro

sellerkey fica no serverless env, nao no frontend.
