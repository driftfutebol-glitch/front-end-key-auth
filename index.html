Painel Gerar Keys

Painel hospedado no seu dominio

Gerar e validar licencas com backend proprio.

Este painel web fala com o seu backend privado e nao precisa expor `SellerKey` no navegador. O controle de geracao e validacao fica no seu proprio servidor.

Abrir Painel Gerar Licencas

Arquitetura recomendada

1. Frontend online no seu subdominio

2. Backend PHP + MySQL + Redis no seu servidor

3. Painel usa:

/api/generate-key para criacao

/api/validate-key para validacao online

Seguro

`SellerKey` fica no ambiente do servidor, nao no frontend.
